## 伦理审查

槽点很多。

据刘宏伟所说，到23年底为止手术事宜都是他拍板的，但是24年换了新的领（cao）导（tai）班子，成立了新的伦理委员会。

华侨的伦理委员会每3个月开一次（截至2024年7月），对这3个月期间的相关案例进行伦理审查。我的是5月底开的。

由于伦理审查拖了很久，刘宏伟也没能给我一个准确的出结果的时间，因此在提交材料后到手术确定前我的精神状态逐渐变差。

如果以后也都是每3个月开一次的话，可以认为是2/5/8/11月这样子。

刘宏伟本人也吐槽说伦理委员会拖慢进程，因为伦理审查的拖延导致有案例离开华侨前往其他医院，表示3个月开一次是对患者的不尊重（可能导致患者的精神状态恶化、导致患者的生涯规划被打乱等）。

至少我的生涯规划就被打乱了——原计划是4-5月手术，9月复学，这样就有三个月的时间进行改证及后续各种事情，但是实际上出院都7月中旬了，因此没法在9月复学，直接这一个学期就寄了。

我也和刘宏伟讲了这个事情，刘宏伟的态度是希望能改成最长每月一次。

2024.7.30更新：

刘宏伟表示时间可能会更长，上海九院那边好像也是三个月。我问411那边呢，刘宏伟说411早就改革了，大家都认可，上传材料直接在网上开会了。

# 入院前准备

## 术前停药和生育力保存问题

刘宏伟表示不需要停药，如果想停的话停一周，我就没停，入院前一晚（day -6）打了一针日雌。

伦理委员会在手术前一天通知我前往泌尿外科进行前列腺液和精液质量检测。
**这是对于手术对象性别烦躁的极大不尊重，也是对跨性别女性的激素治疗效果的无视。**

这既没有尊重手术对象的性别认同和性别烦躁，且手术对象一般都长期进行激素治疗，前列腺液暂且不论，精液质量没什么好测的，因为出不来。如果手术对象确实需要进行生育力保存，也应该提前进行沟通，而不是由伦理委员会**通知**手术对象在手术前夕去泌尿外科进行手淫。

这件事情闹得我很生气，我当时找到我的管床医生，明确表示，这个东西如果不能不做，**我死**。后来刘宏伟通知我，虽然有这个要求，但是**我也有拒绝的权利**。我问他说，那这个东西以后也还会问吗？他说还会，伦理让的。

因此如果有生育力保存的计划，提前跟刘宏伟沟通，避免后续出现这一情况。

## 院内生活

· 病房内有空调，独卫，有淋浴。热水时间为6-9、15-22。

· 早上6点半多，阿姨会来清扫，基本上会吵醒你；护士7点半多会来确认一下你是否醒着，医护人员8点多查房。理论上8-12是治疗时间，但是有时候上午医生没来或者忘了的话，换药和红光治疗等也会拖到下午甚至是午休时间。13-15是午休时间；理论上探访时间是15-21，但是实际上护士不怎么管，别影响别人就行。22点熄灯，床头的小夜灯也比较影响他人，看你在不在乎了（我比较在乎所以我在有病友以后一直很收敛）。

· 半夜护士会巡视，会拿手电筒照你。

· 衣服有病号服。空调可能会比较冷，如果感觉到手脚冰凉务必补衣服。

· 床头有订餐二维码，前一天下午和晚上能订第二天早餐（7点左右），前一天下午到第二天10点能订第二天午餐（11点半左右），当天4点前能订晚餐（5点半左右），具体时间以二维码旁边的说明为准，记不太清了。个人感觉略贵，但是能送到床边，就当是配送费吧，适合自己不方便移动、身边没有人陪护的时候。过了时间以后只有“过时餐”，大概是一荤两素的盒饭，内容随机。
