# 手术预约

## 手术咨询

刘宏伟处要求**在华侨医院有至少1年的就诊经历**才能进行手术，如果不满一年的话会往后推手术时间（作为他的观察期）。

刘宏伟每周二、周五上午出诊整形外科，在华侨医院3号楼2楼218诊室（30r）。

每周二上午，高绿芬会出诊跨性别序列医疗门诊，在华侨医院3号楼2楼219诊室（20r），刘宏伟的隔壁，因此可以在去找高绿芬的时候顺带到隔壁咨询手术事宜。

原则上华侨的手术不用排队（人很少），但是伦理审查可能会比较久，需要做好心理准备。

另外，华侨不是专门做SRS的地方，刘宏伟每周最多只做一台（周三早八）。

还有一个要说的是，刘宏伟本来答应我说6月末给我做，但是后来又通知我改到7月3号，这一点也要做好心理准备。

### 父母知情同意书

尽管根据《性别重置技术临床应用管理规范（2022 年版）》，手术对象需提供**已告知**直系亲属拟行性别重置手术的相关证明，但由于刘宏伟处曾经有过医闹事件，所以刘宏伟要求**父母知情且同意**，并对该文件**进行公证**。

由于我的父母在海外，因此选择了线上远程视频公证。

由我携带父母户口本&结婚证，以及我的出生医学证明，父母在视频对面出示身份证和护照。

2024年3月20日由广州市南沙公证处办理，视频公证和公证书出具共花费2030元。

公证书一式两份，一份在我手上，另一份医院收走留档。

## 伦理审查

槽点很多。

据刘宏伟所说，在23年底手术事宜都是他拍板的，但是24年换了新的领（cao）导（tai）班子，成立了新的伦理委员会。

华侨的伦理委员会每3个月开一次（截至2024年7月），对这3个月期间的相关案例进行伦理审查。我的是5月底开的。

由于伦理审查拖了很久，刘宏伟也没能给我一个准确的出结果的时间，因此在提交材料后到手术确定前我的精神状态逐渐变差。

如果以后也都是每3个月开一次的话，可以认为是2/5/8/11月这样子。

刘宏伟本人也吐槽说伦理委员会拖慢进程，因为伦理审查的拖延导致有案例离开华侨前往其他医院，表示3个月开一次是对患者的不尊重（可能导致患者的精神状态恶化、导致患者的生涯规划被打乱等）。

至少我的生涯规划就被打乱了——原计划是4-5月手术，9月复学，这样就有三个月的时间进行改证及后续各种事情，但是实际上出院都7月中旬了，因此没法在9月复学，直接这一个学期就寄了。

我也和刘宏伟讲了这个事情，刘宏伟的态度是希望能改成最长每月一次。

2024.7.30更新：

刘宏伟表示时间可能会更长，上海九院那边好像也是三个月，我问411那边呢，刘宏伟说411早就改革了，大家都认可，上传材料直接在网上开会了。

我说，还是得跟先进单位多学习。刘说，现在不是现在不学习，现在机关的干部是这样的，能不担责任就不担责任。这个是制度问题，不是技术问题。

## 多学科会诊

在伦理审查通过后，刘宏伟会安排一次多学科会诊，一般是周二早八（实际上要到9点以后手术对象才进会议室）。

可能会和同期的其他trans一起。

多学科会诊由整形外科、精神医学科、妇（产）科、麻醉科和泌尿外科共同召开，主要是排除手术禁忌症，以及对手术的术式、流程等再次进行评估，并和手术对象进行最后确认。

费用为每科200r，五科就是1000r。

多学科会诊开完以后，刘宏伟会安排手术档期，然后通知手术时间。

## 生育力保存问题

伦理委员会在手术前一天通知我前往泌尿外科进行前列腺液和精液质量检测。

这是对于手术对象性别烦躁的极大不尊重，且手术对象一般都长期进行激素治疗，前列腺液暂且不论，精液质量没什么好测的，因为出不来。

如果手术对象确实需要进行生育力保存，也应该提前进行沟通，而不是由伦理委员会**通知**手术对象在手术前夕去泌尿外科进行手淫。

这既没有尊重手术对象的性别认同和性别烦躁，也是对跨性别女性的激素治疗效果的无视。

这件事情闹得我很生气，我当时找到我的管床医生，明确表示，这个东西如果不能不做，**我死**。

后来刘宏伟通知我，虽然有这个要求，但是**我也有拒绝的权利**。我问他说，那这个东西以后也还会问吗？他说还会，伦理让的。

我：……。

不过好消息是至少可以拒绝。
